Windows 安装优化

安装 powershell

  1. 查看当前 powershell 版本，如果不是 7 以上版本则升级为最新版本。
$psversiontable
  1. 下载完后点直接以管理员安装。

安装 scoop

  1. 以管理员身份运行 powershell。

  2. 自定义 scoop 安装目录。

$env:SCOOP='D:\Applications\Scoop'
[Environment]::SetEnvironmentVariable('SCOOP', $env:SCOOP, 'User')
  1. 自定义 scoop 全局安装目录。
$env:SCOOP_GLOBAL='D:\Applications\ScoopGlobal'
[Environment]::SetEnvironmentVariable('ScoopGlobal', $env:SCOOP_GLOBAL, 'Machine')
  1. 自定义 scoop 缓存目录。
$env:SCOOP_CACHE='D:\Applications\ScoopCache'
[Environment]::SetEnvironmentVariable('ScoopCache', $env:SCOOP_CACHE, 'Machine')
  1. 安装 scoop
iwr -useb get.scoop.sh | iex

安装 oh-my-posh

  1. 通过 scoop 安装 oh-my-posh
scoop install https://github.com/JanDeDobbeleer/oh-my-posh/releases/latest/download/oh-my-posh.json
  1. 修改 powershell 配置文件
# 检测有没有配置文件，没有则创建
if (!(Test-Path -Path $PROFILE )) { New-Item -Type File -Path $PROFILE -Force }

# 用 notepad 修改配置文件
notepad $PROFILE
  1. 设置自定义主题
Set-PoshPrompt -Theme  C:\Users\Shi\Documents\PowerShell\oh-my-posh-theme.json
# 'C:\Users\Shi\Documents\PowerShell\oh-my-posh-theme.json' 改成自己主题路径
  1. 在配置文件中添加下面代码
oh-my-posh --init --shell pwsh --config C:\Users\Shi\Documents\PowerShell\oh-my-posh-theme.json | Invoke-Expression
# 'C:\Users\Shi\Documents\PowerShell\oh-my-posh-theme.json' 改成自己主题路径

Set-PSReadLineOption -PredictionSource History
# 启用 powershell 历史命令提示功能
  1. 刷新配置文件或者重启 terminal
. $PROFILE
